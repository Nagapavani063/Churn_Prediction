<div class="section">

<h2> Project Overview</h2>

<p>
The <b>AI-Powered Customer Retention Prediction System</b> is a Machine Learning project developed to analyze telecom customer behavior and predict customer churn.
</p>

<p>
Customer churn refers to customers leaving one telecom service provider and switching to another provider.
</p>

<p>This project focuses on:</p>

<ul>
<li>Data preprocessing</li>
<li>Feature engineering</li>
<li>Feature selection</li>
<li>Data balancing</li>
<li>Model training</li>
<li>Churn prediction using Machine Learning</li>
</ul>

<p>
The system helps telecom companies identify customers who are likely to leave their services, enabling proactive customer retention strategies.
</p>

</div>
